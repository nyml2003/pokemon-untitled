## 产品方向

项目目标是让稳定的战斗规则、大世界和 AI 对战按顺序形成可验证能力。世界应由统一规则驱动，玩家选择能影响局部状态，但不应绕过规则或冻结世界运行。Agent、模型接入、联网、具体剧情、阵营与数值均仍需要独立产品决策，不能由本目录自动授权。

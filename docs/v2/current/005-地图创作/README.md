# 005 地图创作

> 分类：现状；最后核对：2026-07-20

本模块说明地图项目、瓦片语义、原生编辑器、CLI 和两种地图表示。它不定义游戏世界规则本身。

| 顺序 | 页面 | 内容边界 |
| ---: | --- | --- |
| 001 | [地图模型](001-地图模型.md) | `MapProject`、图块、碰撞、事件、角色和编辑历史。 |
| 002 | [地图编辑状态机](002-地图编辑状态机.md) | `EditorModel`、意图、controller、virtual command 与效果。 |
| 003 | [原生编辑器](003-原生编辑器.md) | Winit 输入、工作台投影、渲染和 JSON 保存。 |

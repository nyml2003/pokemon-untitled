# 004 游戏运行时

> 分类：现状；最后核对：2026-07-20

本模块说明单机游戏从启动到帧提交的当前实现。它覆盖 `game-host` 装配的游戏链路，不描述地图编辑器的工作流。

| 顺序 | 页面 | 内容边界 |
| ---: | --- | --- |
| 001 | [状态所有权](001-状态所有权.md) | `GameSession`、世界、战斗和表现状态。 |
| 002 | [启动与装配](002-启动与装配.md) | 资产、地图、叙事脚本、窗口和 native target 的创建。 |
| 003 | [输入与逻辑推进](003-输入与逻辑推进.md) | Winit 输入归一化、表现动作和逻辑 tick。 |
| 004 | [场景投影与帧提交](004-场景投影与帧提交.md) | `GameSnapshot`、`SceneFrame`、`FramePlan` 与 WGPU。 |
| 005 | [失败处理与恢复](005-失败处理与恢复.md) | 启动、投影、帧计划、表面和提交失败的责任边界。 |
| 006 | [游戏会话状态机](006-游戏会话状态机.md) | 世界/战斗切换、命令守卫、观察缓存和 tick。 |
| 007 | [帧计划与提交协议](007-帧计划与提交协议.md) | 多 pass 计划、资源解析、surface 策略和文本覆盖层。 |
